# 纸笔游戏--贪吃蛇
Pen and Pencil Gluttonous snake,you can print it on A4 paper. support two players
纸笔游戏--贪吃蛇，打印出来，支持两个人玩。

# Introduction
1.两人轮流画直线，直线不能相交
2.可以选择无障碍和有障碍的图
3.自动生成A4纸张大小的页  
4.打印就能玩了。
